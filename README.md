# Deteksi Ayam Mati Menggunakan YOLO11

## Dataset
[Deteksi Ayam Dataset](https://universe.roboflow.com/daffaadrn/deteksi-ayam-pky9s)

Dataset ini berisi gambar ayam mati yang digunakan untuk melatih model deteksi objek YOLO11.

## Streamlit Web App
[Deteksi Ayam Web App](https://deteksi-ayam.streamlit.app/)

Akses aplikasi **Deteksi Ayam** yang dibuat menggunakan Streamlit untuk mendeteksi ayam mati secara otomatis.
